# Overview

This project demonstrates a cloud-based Security Operations Center (SOC) architecture deployed on AWS using multiple open-source security tools.

The project simulates real-world cyberattacks against a vulnerable application while providing:

- Centralized SIEM monitoring
- Intrusion Detection
- Web Application Firewall protection
- Threat Intelligence integration
- Automated alerting
- SOAR automation workflows
